# valico-backend

Valico backend with NodeJs, Adonis (Almost a Laravel in NodeJS), and JWT session so i can play with multiple fronts (stateless session)

MYSQL


```bash
npm install
```

.env (put an accesible database params)
```bash
cp .env.example .env
```

Migrations
```js
adonis migration:run
```

```bash
adonis key:generate
```

### Run & Build

Dev
```bash
npm run dev
```
